Módulo principal#

Esta es la documentación para el módulo calculadora.py el cual contiene el programa principal que realiza las diferentes operaciones. Este se compone de dos clases: calculadora y calculadora_avanzada.

class calculadora.calculadora#

Bases: object

Esta primera clase está diseñada para una calculadora simple que realiza: suma, resta, multiplicación y división.

cociente(a: float, b: float) → float#

Esta función divide dos números a y b.

Parámetros:

  • a (float) – primer número (numerador).

  • b (float) – segundo número (denominador).

Devuelve:

El cociente de a y b.

Tipo del valor devuelto:

float

Muestra:

ValueError – si b == 0 \((b = 0)\).

producto(a: float, b: float) → float#

Esta función multiplica dos números a y b.

Parámetros:

  • a (float) – primer número.

  • b (float) – segundo número.

Devuelve:

El producto de a y b.

Tipo del valor devuelto:

float

resta(a: float, b: float) → float#

Esta función resta dos números a y b.

Parámetros:

  • a (float) – primer número.

  • b (float) – segundo número.

Devuelve:

La resta de a y b.

Tipo del valor devuelto:

float

suma(a: float, b: float) → float#

Esta función suma dos números a y b.

Parámetros:

  • a (float) – primer número.

  • b (float) – segundo número.

Devuelve:

La suma de a y b.

Tipo del valor devuelto:

float

class calculadora.calculadora_avanzada#

Bases: object

Esta segunda clase está diseñada para una calculadora avanzada que realiza: potencia, raíz cuadrada y módulo.

modulo(a: float, b: float) → float#

Esta función calcula el módulo de dos números a y b que es el residuo de la división entera entre a y b.

Parámetros:

  • a (float) – primer número (numerador).

  • b (float) – segundo número (denominador).

Devuelve:

el módulo de a y b \((a\mod{b})\).

Tipo del valor devuelto:

float

Muestra:

ValueError – si b == 0 \((b = 0)\).

potencia(a: float, b: float) → float#

Esta función calcula la potencia de un número a elevado a un número b.

Parámetros:

  • a (float) – primer número (base).

  • b (float) – segundo número (exponente).

Devuelve:

El resultado de a elevado b \((a^b)\).

Tipo del valor devuelto:

float

Muestra:

ValueError – si a == 0 and b == 0 \((a = b = 0)\).

raiz(a: float) → float#

Esta función calcula la raíz cuadrada de un número a.

Parámetros:

a (float) – número al que se le aplica la función raíz cuadrada.

Devuelve:

El resultado de la raíz de a \((\sqrt{a})\).

Tipo del valor devuelto:

float

Muestra:

ValueError – si a es negativo.

calculadora.main()#

Esta es la función que ejecuta las operaciones de la calculadora y la calculadora avanzada.

De inicio indica la calculadora que usará, así, solicita al usuario los Parámetros para ejecutar las operaciones.

Finalmente, muestra los resultados obtenidos.
