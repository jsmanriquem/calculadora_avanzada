Módulo de testing#

Esta es la documentación para el módulo de test test_calculadora.py, el cual, como su nombre lo indica, se encarga de revisar que el módulo principal calculadora.py funcione correctamente.

test_calculadora.calc_avanzada = <calculadora.calculadora_avanzada object>#

Se instancia el objeto para la calculadora avanzada.

test_calculadora.calc_simple = <calculadora.calculadora object>#

Se instancia el objeto para la calculadora simple.

test_calculadora.test_cociente()#

Pruebas a realizar para el cociente:

  • calc_simple.cociente(100, 5) debe ser igual a 20 \((100 \div 5 = 20)\).

  • calc_simple.cociente(3, 2) debe ser igual a 1.5 \((3 \div 2 = 1.5)\).

  • Verifica que se lance un error al dividir por cero.

test_calculadora.test_modulo()#

Pruebas para el módulo:

  • calc_avanzada.modulo(20, 6) debe ser igual a 2 \((20\mod{6} = 2)\).

  • calc_avanzada.modulo(18, 5) debe ser igual a 3 \((18\mod{5} = 3)\).

  • Verifica que se lance un error al calcular el módulo por cero.

test_calculadora.test_potencia()#

Pruebas para la potencia:

  • calc_avanzada.potencia(5, 2) debe ser igual a 25 \((5^{2} = 25)\).

  • calc_avanzada.potencia(4, 3) debe ser igual a 64 \((4^{3} = 64)\).

  • calc_avanzada.potencia(-2, 3) debe ser igual a -8 \(((-2)^{3} = -8)\).

test_calculadora.test_producto()#

Pruebas a realizar para el producto:

  • calc_simple.producto(7, 6) debe ser igual a 42 \((7 \cdot 6 = 42)\).

  • calc_simple.producto(-3, 7) debe ser igual a -21 \((-3 \cdot 7 = -21)\).

  • calc_simple.producto(1.1, 2) debe ser igual a 2.2 \((1.1 \cdot 2 = 2.2)\).

test_calculadora.test_raiz()#

Pruebas para la raíz cuadrada:

  • calc_avanzada.raiz(25) debe ser 5 \((\sqrt{25} = 5)\).

  • calc_avanzada.raiz(144) debe ser 12 \((\sqrt{144} = 12)\).

  • Verifica que se lance un error al calcular la raíz cuadrada de un número negativo.

test_calculadora.test_resta()#

Pruebas a realizar para la resta:

  • calc_simple.resta(100, 50) debe ser igual a 50 \((100 - 50 = 50)\).

  • calc_simple.resta(-5, 10) debe ser igual a -15 \(((-5) - 10 = -15)\).

  • calc_simple.resta(3.2, 1.2) debe ser igual a 2 \((3.2 - 1.2 = 2)\).

test_calculadora.test_suma()#

Pruebas a realizar para la suma:

  • calc_simple.suma(0, 7) debe ser igual a 7 \((0 + 7 = 0)\).

  • calc_simple.suma(-10, -5) debe ser igual a -15 \(((-10) + (-5) = -15)\).

  • calc_simple.suma(1.5, 2.5) debe ser igual a 4 \((1.5 + 2.5 = 4)\).
